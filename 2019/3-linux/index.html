Linux用户登陆和注销

Written by Bing with ♥ on November 9, 2019 in linux

​ 登录时尽量少用root账户，因为它是系统管理员，最大权限，避免操作失误。可以用普通用户登录，用”su-用户名”命令来切换成系统管理员身份

​ 在终端输入logout即可注销用户

​ logout注销指令在图形运行级别无效，在运行级别3下有效
