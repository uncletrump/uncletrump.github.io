Linux开机&重启命令

Written by Bing with ♥ on November 9, 2019 in linux 
shutdown -h now	#立刻关机
shutdown -h 1	#一分钟后关机
shutdown -r now #现在重新启动计算机
halt #关机
reboot #现在重新启动计算机
sync #把内存数据同步到磁盘，关机或者重启前执行
