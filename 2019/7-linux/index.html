Linux文件目录指令

Written by Bing with ♥ on November 9, 2019 in linux

常用命令

pwd

pwd #显示当前工作目录的绝对路径

ls

ls [options] [目录或者文件] #显示当前目录下的所有文件和目录
	-a:#显示当前目录的所有文件和目录，包括隐藏的
	-l:#以列表形式显示信息
	-h:#显示文件大小时，以k，m，G单位显示

cd

cd [path] #切换工作目录

mkdir

mkdir [options] [目录] #创建文件夹
	-p #创建多级目录 #e.g. mkdir -p /xx/xx/xx

rmdir

rmdir [options] [目录] #删除空目录，有内容时无法删除

rm

rm [options] [目录或者文件] #删除文件或目录
	-r #表示递归
	-f #表示强制删除，不需要询问

touch

touch [文件] #创建一个空文件，若文件存在，表示刷新文件的修改时间

cp

cp [options] source[源] dest[目的文件] #指令拷贝文件到指定目录
	-r #递归复制整个文件夹
	#注意：\cp表示强制覆盖不提示

mv

mv #移动文件或者重命名
#操作时，如果在同一目标，表示改名，否则表示移动

cat

cat #查看文件内容
	cat [options] [filename]
		-n #显示行号
	#如果需要显示下一行，输入enter
	#如果需要翻页，输入空格键
	#如果需要退出，输入q
	#cat只能浏览文件，不能修改文件，为了浏览方便，一般会带上管道命令 | more
		#e.g.
		#cat xxx.txt | more

more

more [文件]
#more指令是一个基于VI编辑器的文本过滤器，它以全屏幕的方式按页显示文本文件的内容。more指令中内置了若干快捷键
快捷键
操作 功能说明
空白键 (space) 代表向下翻一页
Enter 代表向下翻『一行』
q 代表立刻离开 more ，不再显示该文件内容
Ctrl+F 向下滚动一屏
Ctrl+B 返回上一屏
= 输出当前行的行号
:f 输出文件名和当前行的行号

less

less指令用来分屏查看文件内容，它的功能与more指令类似，但是比more指令更加强大，支持各种显示终端。less指令在显示文件内容时，并不是一次将整个文件加载之后才显示，而是根据显示需要加载内容，对于显示大型文件具有较高的效率。

less [filename]
快捷键
操作 功能说明
Space 向下翻动一页
Pagedown 向下翻动一页
Pageup 向上翻动一页
/字串 向下搜寻[字串]功能 n:向下查找 N:向上查找
?字串 向上搜寻[字串]功能 n:向上查找 N:向下查找
q 离开less这个程序

> & >>

> #输出重定向
>> #追加
ls -l > filename #列表内容写入file中（覆盖写）
ls -al >> filename #列表内容追加到文件file的末尾
cat file1 > file2 #将文件1内容覆盖到文件2
echo "内容" >> file #将内容追加到file末尾

echo

输出内容到控制台

echo [options] [输出内容]
#e.g.
#echo $PATH

head

head用于显示文件开头部分内容，默认情况下显示文件的前10行内容

head 文件 #（查看文件头10行内容）
head -n 5 文件#(查看文件头5行内容，5可以是任意行数)

tail

tail用于输出文件中尾部的内容，默认情况下tail指令显示文件的后10行内容

tail 文件 #（查看头10行文件）
tail -n 5 文件 #（查看头5行文件，5可以是任意数）
tail -f 文件 #（监控文件的所有更新）

ln

软链接也成为符号链接，类似于windows里的快捷方式，主要存放了链接其他文件的路径

ln -s [原文件或目录] [软链接名] （功能描述：给原文件创建一个软链接）
#当我们使用pwd指令查看目录时，仍然看到的是软链接所在目录

history

查看已经执行过历史命令,也可以执行历史指令

history #显示所有的历史命令
history 10 #显示最近使用过的10个指令
!5 #执行历史编号为5的指令
