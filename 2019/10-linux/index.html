Linux压缩和解压类

Written by Bing with ♥ on November 10, 2019 in linux

gzip/gunzip

gzip用于压缩文件，只能将文件压缩为*.gz类型，压缩后不会保留原文件

gunzip用于解压文件

#e.g.
gzip /home hello.txt
gunzip /home hello.txt.gz

zip/unzip

zip [options] XXX.zip
	-r #递归压缩，即压缩目录
unzip [option] XXX.zip
	-d [目录] #指定解压后文件的存放目录
	

#e.g.
#将 /home下的 所有文件进行压缩成 mypackage.zip
zip  -r   mypackage.zip  /home/
#将 mypackge.zip 解压到 /opt/tmp 目录下
unzip –d  /opt/tmp  mypackage.zip   #[如果/opt/tmp 不存在，会自动创建]

tar

tar 指令 是打包指令，最后打包后的文件是 .tar.gz 的文件。 [可以压缩，和解压]

tar  [选项]  XXX.tar.gz  打包的内容/目录
	-c #产生.tar打包文件
	-v #显示详细信息
	-f #指定压缩后的文件名
	-z #打包同时压缩
	-x #解压.tar文件
	
#e.g.
#压缩多个文件
tar  -zcvf  a.tar.gz  a1.txt  a2.txt
#压缩/home 的文件夹
tar  -zcvf   myhome.tar.gz   /home/
#将 a.tar.gz  解压到当前目录
tar -zxvf  a.tar.gz 
#将myhome.tar.gz  解压到 /opt/tmp2目录下
tar -zxvf myhome.tar.gz  -C  /opt/tmp2  [注意 /opt/tmp2 事先需要创建好]
