使用maven搭建spring开发环境

Written by Bing with ♥ on March 1, 2020 in spring

  1. 倒入spring核心jar包

    • spring-context

    • spring-core

    • spring-beans

    • spring-expression

  2. 创建javabean

    public class HelloWord {
   
   private String name;
   
   public void setName(String name) {
       this.name = name;
   }
   
   public void hello() {
       System.out.println("Hello" + name);
   }
}

  3. 类路径下创建applicationContext.xml

    <?xml version="1.0" encoding="UTF-8"?>
<beans xmlns="http://www.springframework.org/schema/beans"
      xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
      xsi:schemaLocation="http://www.springframework.org/schema/beans http://www.springframework.org/schema/beans/spring-beans.xsd">
   
   <!--通过全类名用反射的方式在IOC容器中创建bean，必须提供无参构造-->
   <!--id为Bean的名称，在IOC容器中必须是唯一的，
   若没有指定，spring将自动把类名作为Bean的名字，
   可以指定多个名字，用逗号、分号、空格隔开-->
   <bean id="helloWord" class="org.ouhei.spring.beans.HelloWord">
       <property name="name" value="Java"></property>
   </bean>
</beans>

  4. 建立测试类

    public class TestMain {
   public static void main(String[] args) {
   
       // 创建Spring IOC容器，并会调用构造方法，通过set方法对属性进行赋值
       ApplicationContext ac=new ClassPathXmlApplicationContext("applicationContext.xml");
   
       // 从容器中获取bean对象
       HelloWord helloWord = (HelloWord) ac.getBean("helloWord");
   
       helloWord.hello();
   }
}
