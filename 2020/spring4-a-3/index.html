依赖注入的方式

Written by Bing with ♥ on March 1, 2020 in spring

Spring 支持 3 种依赖注入的方式

  • 属性注入(setter)
  • 构造器注入
  • 工厂方法注入（很少使用，不推荐）

属性注入

属性注入即通过 setter 方法注入Bean 的属性值或依赖的对象。

属性注入使用 元素, 使用 name 属性指定 Bean 的属性名称，value 属性或 子节点指定属性值

参考：使用maven搭建spring开发环境

构造方法注入

必须提供有参构造方法

通过构造方法注入Bean 的属性值或依赖的对象，它保证了 Bean 实例在实例化后就可以使用。

构造器注入在 元素里声明属性, 中没有 name 属性

public class Student {

    private String name;

    private Integer num;

    private Double point;

    public String getName() {
        return name;
    }

    public Integer getNum() {
        return num;
    }

    public Double getPoint() {
        return point;
    }

    public void setName(String name) {
        this.name = name;
    }

    public void setNum(Integer num) {
        this.num = num;
    }

    public void setPoint(Double point) {
        this.point = point;
    }

    public Student() {
    }

    public Student(String name, Integer num, Double point) {
        this.name = name;
        this.num = num;
        this.point = point;
    }

    @Override
    public String toString() {
        return "Student{" +
                "name='" + name + '\'' +
                ", num=" + num +
                ", point=" + point +
                '}';
    }
}

public class TestMain {
    public static void main(String[] args) {

        // 创建Spring IOC容器，并会调用构造方法，通过set方法对属性进行赋值
        ApplicationContext ac=new ClassPathXmlApplicationContext("applicationContext.xml");

        // 从容器中获取bean对象
        Object student1 = ac.getBean("student1");
        System.out.println(student1.toString());

        Object student2 = ac.getBean("student2");
        System.out.println(student2.toString());
    }
}

<?xml version="1.0" encoding="UTF-8"?>
<beans xmlns="http://www.springframework.org/schema/beans"
       xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
       xsi:schemaLocation="http://www.springframework.org/schema/beans http://www.springframework.org/schema/beans/spring-beans.xsd">

    <!--类型和索引可以混合使用-->
    <!--按类型匹配-->
    <bean id="student1" class="org.ouhei.spring.beans.Student">
        <constructor-arg value="小明" type="java.lang.String"/>
        <constructor-arg value="13014" type="java.lang.Integer"/>
        <constructor-arg value="521.34" type="java.lang.Double"/>
    </bean>

    <!--按索引配置-->
    <!--可以用子标签value进行赋值，输出特殊字符时<![CDATA[]]>-->
    <bean id="student2" class="org.ouhei.spring.beans.Student">
        <constructor-arg index="0">
            <value><![CDATA[<小红>]]></value>
        </constructor-arg>
        <constructor-arg value="13015" index="1"/>
        <constructor-arg type="java.lang.Double">
            <value>692.34</value>
        </constructor-arg>
    </bean>
</beans>
