spEL

Written by Bing with ♥ on March 8, 2020 in spring

Spring表达式语言：SpEL

整数

#{5}

小数

#{89.7}

科学计数法

#{1e4}

Boolean

#{false}

String

#{'Chuck'}或者#{"Chuck"}

引用bean，属性和方法

引用其他对象

#{object}

引用其他对象的属性

#{object.property}

调用其他方法，还可以链式操作

#{object.toString().toUpperCase()}

算术运算符

+, -, *, /, %, ^

加号作为字符串拼接符

#{object.name+' '+object.company}

比较运算符

<, >, ==, <=, >=, lt, gt, eq, le, ge

逻辑运算符

and, or, not, |

if-else 运算符：?: (ternary), ?: (Elvis)

#{object.name=='asuka'?'aoi':'kirara'}

if-else 的变体

#{object.name=='asuka'?'aoi'}

正则表达式：matches

#{object.email matches '[xxxxxx]'}

调用静态方法或静态属性：通过 T() 调用一个类的静态方法，它将返回一个 Class Object，然后再调用相应的方法或属性

#{T(java.lang.Math).PI}
